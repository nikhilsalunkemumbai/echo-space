# Engine

The Volatility Echo simulation engine is not published here yet.

**Why:** The engine contains the full implementation of the crisis attribution mechanism — the Volatility Echo. Publishing this before the RCT study completes would risk contaminating the study: participants who find the repository before playing could understand exactly how the system works, which would invalidate the primary behavioral outcome measures (decision latency, action selection under uncertainty).

**What is available now:**
- Full technical description in [docs/whitepapers/](../docs/whitepapers/)
- Architecture overview in [ARCHITECTURE.md](../ARCHITECTURE.md)
- Calibration code and results in [research/](../research/)
- The locked parameters are documented in [ARCHITECTURE.md](../ARCHITECTURE.md)

**When it will be published:**
The full engine source code will be released here as open-source supplementary material when the results paper is submitted, following pre-registered RCT completion. Pre-registration is at https://osf.io/PLACEHOLDER.

This is not gatekeeping. It is study integrity.
